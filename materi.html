1. Pengertian Multimedia Desain Grafis

Multimedia desain grafis merupakan proses penyampaian informasi visual menggunakan kombinasi teks, gambar, warna, ilustrasi, dan elemen visual lainnya untuk menyampaikan pesan secara menarik.

2. Fungsi dan Manfaat Desain Grafis

  • 📢 Media komunikasi visual
  • 🛍 Media promosi produk
  • 📚 Media pembelajaran
  • 📰 Media informasi

3. Pengenalan CorelDRAW

CorelDRAW adalah software desain grafis berbasis vektor yang digunakan untuk membuat logo, poster, banner, brosur, ilustrasi, dan berbagai media visual lainnya.

4. Antarmuka CorelDRAW

  • Title Bar
  • Menu Bar
  • Toolbox
  • Property Bar
  • Color Palette
  • Drawing Area
  • Status Bar

5. Tools pada CorelDRAW

🖱 Pick Tool

Digunakan untuk memilih dan memindahkan objek.

✏ Shape Tool

Digunakan untuk mengedit bentuk objek.

⬛ Rectangle Tool

Membuat persegi atau persegi panjang.

⭕ Ellipse Tool

Membuat lingkaran atau oval.

🔤 Text Tool

Untuk membuat teks.

🎨 Fill Tool

Memberi warna pada objek.

6. Gambar Vektor dan Bitmap

Vektor

Tidak pecah saat diperbesar karena berbasis garis dan kurva.

Bitmap

Dapat pecah saat diperbesar karena tersusun dari piksel.

7. Teori Warna

  • RGB (Red, Green, Blue)
  • CMYK (Cyan, Magenta, Yellow, Black)
  • Warna Primer
  • Warna Sekunder

8. Tipografi dalam Desain

Tipografi adalah seni mengatur tulisan agar mudah dibaca dan memiliki nilai estetika.

9. Layout dan Komposisi

  • Balance (Keseimbangan)
  • Contrast (Kontras)
  • Alignment (Perataan)
  • Hierarchy (Hierarki Visual)

10. Langkah Membuat Desain di CorelDRAW
